# LCM Recommender Request Packet — 2026-07-04

## BLUF
Use these short messages to request the two signed recommendation PDFs. Recommenders should edit templates into their own voice and sign only what they can personally attest to.

## Universal ask text

```text
I’m applying for a Livestock Conservancy Microgrant for Arapawa goat conservation infrastructure at Urban Ark Conservation.

Would you be willing to sign a short recommendation letter if I send you a draft template? Please edit it into your own voice, remove anything you cannot personally attest to, place it on your letterhead if you have one, sign it, and return it as a PDF.

The grant supports quarantine/buck housing/handling infrastructure so we can safely add an unrelated Arapawa bloodline and strengthen the conservation breeding program.
```

## Letter 1 — conservation / breed / peer

Use with an Arapawa registry contact, Arapawa breeder, conservation peer, or agricultural professional.

```text
The strongest points to cover are:
- how you know Chad/UAC;
- whether UAC is doing real heritage-breed conservation work;
- why Arapawa goats matter as a Critical breed;
- why adding a second unrelated bloodline and quarantine/breeding infrastructure is important;
- whether you recommend UAC for the Microgrant.
```

## Letter 2 — vet / veteran / education-program partner

Use with the attending veterinarian, a program partner, or a veteran participant/leader who has directly seen the farm’s work.

```text
The strongest points to cover are:
- how you know Chad/UAC;
- what you have personally observed at the farm;
- whether UAC handles livestock and/or veteran agricultural programming responsibly;
- how the Arapawa conservation work supports a broader public/community mission;
- whether you recommend UAC for the Microgrant.
```

## Two-veteran fallback rule
Two veteran letters can be used only if both can truthfully speak to UAC’s conservation work and no stronger conservation/vet signer is available in time. This is Plan B, not the preferred proof stack.

## Return format
- Signed PDF preferred.
- Letterhead preferred but not required unless the grant portal requires it.
- Must include signer name, title/relationship, contact info, and date.
