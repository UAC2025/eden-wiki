# Grant Submission Approval & Verification Checklist

> **DRAFT CONTROL DOCUMENT — NOT LEGAL/TAX ADVICE.** S-1 may use this internally as a control framework. External use, donor sends, grant submissions, filings, or official adoption require Commander approval and appropriate reviewer input.

## Workflow

```text
opportunity found
→ eligibility checked
→ draft written
→ attachments gathered
→ Commander review
→ Commander approval
→ credential authorization
→ submit
→ verify confirmation
→ archive packet
```

## Pre-submit checklist

- [ ] Grant name and funder recorded.
- [ ] Deadline recorded.
- [ ] Eligibility confirmed or flagged.
- [ ] UAC entity information verified.
- [ ] Budget/totals source-backed.
- [ ] Attachments complete.
- [ ] Required signatures identified.
- [ ] Commander approval recorded for this specific grant.
- [ ] Credential/access authorization recorded for this submission.

## Post-submit checklist

- [ ] Submission confirmation captured.
- [ ] Confirmation number/email/PDF archived.
- [ ] Follow-up deadline created.
- [ ] Reporting obligations entered into S-1/TaskAI ledger.
- [ ] Commander notified of verified result.
