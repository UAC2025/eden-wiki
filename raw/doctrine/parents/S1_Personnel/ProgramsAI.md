provides · Learn — prior lessons (what programming produced outcomes) ·
Decide — design/schedule/record within scope · Act — produce the curriculum,
program packet, schedule, or outcomes report a human uses · Adapt —
self-verify participant data against sources; outcomes claims cite evidence ·
Repeat per mission block.
